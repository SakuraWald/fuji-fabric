\sss{delay}
\ssss{Purpose}
Delay command allows you to execute a command in specific seconds.

\ssss{Command}
\sssss{/delay}

\ssss{Example}
\begin{example}{A basic usage}
    \ic{/delay 3 say three seconds passed}
\end{example}

\begin{example}{A nested delay}
    \ic{/delay 1 delay 2 delay 3 say 6 seconds passed.}
\end{example}
