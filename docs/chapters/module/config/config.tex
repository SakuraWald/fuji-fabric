\LevelOne{config}

\LevelTwo{Purpose}
This module provides the root command \ic{/fuji}, which includes some operations on fuji itself.

\LevelTwo{Command}
\LevelThree{/fuji reload}
Reload all configuration files and all modules.

\begin{note}{Module itself can't be hot reloaded}
    After you \ttt{enable} or \ttt{disable} a module, you must \ttt{restart} your server.
\end{note}
